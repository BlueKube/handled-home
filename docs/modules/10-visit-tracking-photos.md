# Module 10: Visit Tracking & Photos

## Scope
Real-time visit status for customers, before/after photo review, line-item ratings.

## Tables
- `visit_ratings` — job_id, user_id, rating (1-5), comment, created_at

## Key User Stories
- As a customer, I can track my Service Day in real-time
- As a customer, I can review before/after photos
- As a customer, I can rate each service line item

## Dependencies
- Module 09 (job execution)

## Acceptance Criteria
- [ ] Real-time status updates visible to customer
- [ ] Photos displayed per job
- [ ] Rating submission works
- [ ] Ratings feed into provider scorecard

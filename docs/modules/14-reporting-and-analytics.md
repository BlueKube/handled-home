# Module 14: Reporting & Analytics

## Scope
MRR dashboards, churn analysis, provider utilization, density metrics, audit logs.

## Tables
- `audit_logs` — actor_id, action, resource_type, resource_id, metadata (jsonb), created_at

- As an admin, I can view MRR and churn metrics
- As an admin, I can see provider utilization by zone
- As an admin, I can review density dashboards
- As an admin, I can audit all system actions

# design-guidelines.md

## Emotional Tone

Feels like a calm concierge — confident, kind, predictable.

---

## Typography

Primary: Inter  
H1: 32 / 700  
H2: 24 / 700  
H3: 18 / 650  
Body: 16 / 450–500  
Caption: 13  

Line-height ≥1.5x  

---

## Color System

Primary: #0B1F3B  
Accent: #14B8A6  
Surface: #F7F8FA  
Card: #FFFFFF  
Border: #E6E8EE  

Success: #16A34A  
Warning: #F59E0B  
Error: #DC2626  

WCAG AA compliance required.

---

## Spacing

8pt grid system  
Tap targets ≥44px  
One primary CTA per screen  

---

## Motion

150–300ms transitions  
Gentle easing  
No aggressive animations  

---

## Voice

Calm, competent, kind.  
Never blame users.  
Reinforce promise: “Your home is handled.”

---

## Accessibility

- Semantic headings
- Visible focus states
- Proper contrast ratios
- Clear error messaging
- Keyboard navigation for admin

---

## Design Integrity Review

Visual tone aligns with operational clarity.  
Proof-of-work (photos) should always feel like reassurance.

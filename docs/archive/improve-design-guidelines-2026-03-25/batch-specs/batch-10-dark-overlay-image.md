# Batch 10: Dark Overlay & Image Treatment

## Phase
Phase 3: Dark Mode Polish

## Scope
1. Add `dark:bg-black/70` to dialog, sheet, drawer overlays (increase opacity in dark mode)
2. Add CSS utility `.dark-image-filter` for photos: `brightness(0.85)` in dark mode
3. Verify toast readability in dark mode — add `dark:border-border` if needed

## Acceptance criteria
- [ ] Dialog overlay has dark:bg-black/70
- [ ] Sheet overlay has dark:bg-black/70
- [ ] Drawer overlay has dark:bg-black/70
- [ ] .dark-image-filter CSS utility exists
- [ ] Toast has dark:border-border
- [ ] npm run build passes
